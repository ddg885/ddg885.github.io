# Navy Bonus Optimization Macro

This directory contains a VBA module `NavyBonusOptimization.bas` that implements
a macro for estimating sailor bonus requirements and assigning bonus tiers based
on available budgets. The macro expects the following worksheets in the active
workbook:

- **Inventory** – rating inventory, billet requirements, category (NAT, PS or
  BOTH), allocation percentages and the last five years of retention rates.
- **Tier Mapping** – list of bonus tiers for NAT and PS. Columns: Category,
  Tier name and Bonus amount.
- **Budget** – budget caps for NAT and PS in cells `B2` and `C2` respectively.
- **Raw Output** – output table generated by the macro.
- **Optimal Tier** – table that shows the calculated tier ranges.

Run the `OptimizeBonuses` macro to populate the `Raw Output` and
`Optimal Tier` sheets. The macro now validates that all required
worksheets exist and outputs separate rows for NAT and PS bonus
allocations. Adjust the sheet names if they differ in your workbook.
